Technical Details

Root Cause:

The leaderboard displays cached totalScore field from the Firebase users collection, while the player stats modal calculates scores in real-time by summing all activities from the activities collection. These two data sources were out of sync.

Solution:
  • Detection: New audit tool compares stored vs calculated scores for all users
  • Prevention: Enhanced sync function with better error handling and race condition protection
  • Recovery: Both manual (user refresh button) and admin (bulk fix) tools available
